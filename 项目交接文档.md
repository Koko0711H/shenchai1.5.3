# 深柴动力 500kW 产品子站交接文档

更新日期：2026-07-22

## 项目定位

- 产品：500kW 开架式柴油发电机组
- 技术：React + Three.js + Vite
- 默认语言：英文，可切换中文
- 源码入口：`src/App.jsx`、`src/App.css`
- 静态资源：`public/`
- 线上部署目录：`dist/`

## 当前页面能力

- 深柴动力统一蓝色品牌、字体与顶部导航。
- 3D 产品滚动展示与产品站切换入口。
- 全屏淡蓝色光晕随页面进度平滑变化，不显示生硬光束路径。
- 粒子、按钮、导航和强调色已移除原橙色体系。
- 最后一屏为询价页面，使用 `public/inquiry-factory.png`，带半透明遮罩和由下向上的进场动效。
- 页尾和获取报价入口已接入统一主站导航体系。

## 构建与部署

```powershell
pnpm install
pnpm run build
```

构建后直接部署 `dist/`，不要只上传 `src/`。2026-07-22 已重新构建并验证。

## 维护注意

- 修改品牌文案时同步检查中英文对象。
- 替换询价背景时保持文件名 `inquiry-factory.png`，或同步修改 CSS 路径。
- Three.js 主包较大，构建时会出现 chunk 体积提示，但不影响当前产物使用。
